# 🚀 Kunal Shrivastava — Personal Portfolio

A high-end, interactive personal portfolio website built for
a B.Tech AI & Data Science student at JECRC Foundation, Jaipur.

## ✨ Features

- 🌐 **Cyberpunk Glassmorphism** dark theme
- 🎬 **Boot sequence loading screen** with matrix rain animation
- 🖱️ **Custom cursor** with white glow trail effect
- 🎭 **3D Cyberpunk Avatar** — hover to reveal real photo
- 🌌 **Interactive canvas background** — particles, moving
  grid, mouse repulsion
- ⚡ **Smooth scroll** navigation on all buttons
- 🃏 **6 Project cards** with unique animated SVG backgrounds
- 📄 **Resume download** button (embedded PDF)
- 🎨 **Color-coded hover effects** on every section card
- 📱 **Fully responsive** for mobile and desktop

## 🛠️ Tech Stack

- **React.js** + **Vite**
- **Lucide React** — icons
- **CSS Animations** — keyframes, glassmorphism, canvas API
- **Vanilla Canvas API** — particle system & background

## 📂 Sections

1. **Hero** — Typing effect, animated stats, 3D avatar
2. **About** — Bio, quick facts, education, interests
3. **Skills** — 20+ technologies with hover pill effects
4. **Projects** — 6 ML/AI/Full-stack projects with live animations
5. **Experience** — Internship timeline + achievements + certifications
6. **Contact** — Working form + social links + direct contact

## 🚀 Run Locally
```bash
git clone https://github.com/yourusername/kunal-portfolio.git
cd kunal-portfolio
npm install
npm install lucide-react
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
